ZT Megamenu
==========
ZT Megamenu is a module that helps in altering the default menu styles to a
more stylistic megamenu.

You can select any menu to have the megamenu style by selecting that menu from
the given module configuration form.

This module has been designed to used on a Drupal default theme. Please ensure
to set "Maximum number of menu levels to display" of the selected menu to
"Unlimited" for the menu to work.
